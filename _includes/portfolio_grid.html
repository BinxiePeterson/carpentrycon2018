Our Lessons

All Software Carpentry lessons are hosted on GitHub. Repositories are lesson-specific, meaning that lessons such as Unix shell and Programming with Python all have their own separate repositories. You can find our lessons through our website, and either . connect to the public-facing view of individual lessons, or see the repository view. In the repository view, you can create issues, fork the lessons, or submit pull requests for changes. Lessons include community-contributed content.
